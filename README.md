# mps-arch

```
Usage: make RAW|META

  RAW:
     homedir       : essential folders and config files in HOME
     bash          : bash utils
     edit          : THE editor (vim,neovim)
     archives      : archive tools (zip,bzip,etc.)
     transport     : transport tools (wget,curl, etc.)
     aur           : yay for aur installations
     osutils       : some essentials os tools (psmisc,htop,etc.)
     min           : homedir,bash,edit,archives,transport,aur,osutils
     dev           : min + development essentials
     net           : min + network essentials
     txt           : min + texlive-full,pandoc

  META:
     essentials    : Bare minimum (only min)
     standard      : Default mps components (All but txt)
     full          : Al
´´´
